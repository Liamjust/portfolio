Simple portfolio

Files
- index.html — the portfolio page. Edit the About paragraph and replace the placeholder links and titles for the two websites.
- styles.css — minimal styling for layout and responsiveness.

How to use
1. Open `index.html` in your browser (double-click or right-click -> Open with Browser).
2. Edit the About paragraph in the `index.html` file: find the paragraph with id `about-text` and replace the placeholder text.
3. Update the two website cards: change the <a href="#"> to your real URLs, update titles and descriptions, and replace the image src if you have screenshots.

That's it — a tiny, editable portfolio you can customize quickly.
